Web Development Learning Journey 🚀
Мій репозиторій для збору практичних робіт, навчальних проєктів та експериментів з верстки та фронтенд-розробки.

📌 Про проєкт
Цей репозиторій створений для відстеження прогресу у вивченні сучасних веб-технологій. Тут зібрані роботи, що охоплюють шлях від базового HTML до складних адаптивних інтерфейсів.

🛠 Технологічний стек
HTML5 — семантична верстка та структура.

CSS3 — стилізація, Flexbox, Grid, анімації.

Адаптивність — Media Queries, Mobile First підхід.

Інструменти — VS Code, Git, Chrome DevTools.


🚀 Як запустити локально
Щоб переглянути будь-який з проєктів на власному комп'ютері:

Клонуйте репозиторій:

Bash
git clone https://github.com/ваш-нікнейм/назва-репозиторія.git
Перейдіть у папку проєкту:

Bash
cd назва-папки
Відкрийте файл index.html у вашому улюбленому браузері.
