# Лабораторна робота №6 — Варіант 16
## Тема: Форма додавання товару в базу даних

### Структура проєкту
```
lab6_var16/
├── index.html        ← Адаптивна HTML-сторінка (Bootstrap 5)
├── js/
│   └── script.js     ← JavaScript (валідація + таблиця)
├── app.py            ← Flask-сервер + SQLite
├── products.db       ← Файл БД (створюється автоматично)
└── README.md
```

---

### Запуск

1. Встановити Flask (якщо не встановлено):
   ```bash
   pip install flask
   ```

2. Запустити сервер:
   ```bash
   python app.py
   ```

3. Відкрити у браузері: http://127.0.0.1:5000

> База даних `products.db` створюється автоматично при першому запуску.

---

### API Endpoints

| Метод  | URL                     | Опис                        |
|--------|-------------------------|-----------------------------|
| GET    | /api/products           | Отримати всі товари з БД    |
| POST   | /api/products           | Додати товар до БД          |
| DELETE | /api/products/<id>      | Видалити товар за ID        |
| GET    | /api/products/stats     | Статистика (кількість/сума) |
